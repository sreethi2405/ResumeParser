# Resume-Parser
Analyze, score and rank a collection of PDF resumes using machine learning

Merit | Edge Resume Parser and Scorer

REQUIREMENTS:
INSTALL THE PACKAGES WHICH IS PROVIDED IN requirements.txt


USAGE:
-Name of the resume candidate should be in top
-resume should be in pdf format only
-resume should be included in Sample resumes directory


TOTAL MISSION
Our resume parser and scorer use these components to create a  .pdf file of results:
- Name
-Gender
-Email
-  Highest Qualification Degree Details
